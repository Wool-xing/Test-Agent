# 项目导航（按职责分类）

> 本项目按 **通用测试流程 / 平台专项专家 / 支持协议 / 输入加载 / CI 闭环** 5 大维度组织。
> 每个 agent / skill / utils 在下面表格中标注分类，便于快速定位。
