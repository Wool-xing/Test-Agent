# 贡献者公约（Code of Conduct）

> 基于 [Contributor Covenant 2.1](https://www.contributor-covenant.org/zh-cn/version/2/1/code_of_conduct/) 改写。

## 我们的承诺

为营造开放、欢迎、多元、包容、健康的社区，作为贡献者和维护者，我们承诺让参与本项目（Test-Agent 工作流）的每个人都不会受到骚扰，无论年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人形象、种族、宗教或性取向。

## 我们的标准

**鼓励的积极行为**：

- 对他人表现善意和共情
- 尊重不同的意见、观点和经验
- 给予和优雅地接受建设性反馈
- 承担责任、向受影响的人致歉，并从经验中学习
- 关注对整个社区最有利的事，而非个人

**不可接受的行为**：

- 使用色情语言或图像，以及任何形式的性挑逗或挑逗性行为
- 挑衅、侮辱或贬损性评论，以及人身攻击或政治攻击
- 公开或私下骚扰
- 未经明确许可，发布他人的私人信息（如住址、电子邮箱）
- 在专业场合中可被合理地认定为不当的其他行为

## 执行责任

社区维护者负责澄清和执行可接受行为的标准，并对任何不合适、威胁、冒犯或有害的行为采取适当且公平的纠正措施。

社区维护者有权移除、编辑或拒绝违反本公约的评论、提交、代码、wiki 编辑、问题及其他贡献，并在认为适当时通报禁止理由。

## 适用范围

本公约适用于所有社区场所，也适用于个人在公共场合代表本社区的情况。

代表本社区包括：使用官方电子邮箱、通过官方社交媒体账户发布、在线上或线下活动中作为指定代表参与。

## 执行

辱骂、骚扰或其他不可接受行为可向负责执行的社区维护者举报：

-**GitHub Issue**（公开问题）：[New Issue](https://github.com/Wool-xing/Test-Agent/issues/new) 加 `code-of-conduct` 标签
-**私下举报**：通过 [GitHub Security Advisory](https://github.com/Wool-xing/Test-Agent/security/advisories/new) 私密通道（推荐，仅维护者可见）

所有投诉都将被迅速、公正地审查和调查。

社区领导者有义务为举报者保密。

## 执行准则

社区维护者将遵循以下《社区影响指南》来确定违反本公约的行为后果：

### 1. 纠正

**社区影响**：使用不恰当的语言或被认为不专业、不受社区欢迎的行为。

**后果**：来自社区维护者的私下书面警告，提供违规说明、解释行为不当的原因。可能要求公开道歉。

### 2. 警告

**社区影响**：单一事件或一系列行动构成的违规。

**后果**：警告并附后续行为后果。在指定时段内禁止与相关人员互动，包括与公约执行人员未经请求的互动。违反这些条款可能导致暂时或永久封禁。

### 3. 暂时封禁

**社区影响**：严重违反社区准则，包括持续不当行为。

**后果**：在指定时段内禁止与社区进行任何互动或公开交流。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区准则的模式，包括持续不当行为、骚扰、攻击或贬损某类人。

**后果**：永久禁止在社区内进行任何形式的公开互动。

## 归属
