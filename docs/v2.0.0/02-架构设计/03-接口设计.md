### 1.3 BaseExecutionEnv（保持现有接口）
```python
@dataclass
class ExecResult:
    ok: bool
    stdout: str
    stderr: str

class BaseExecutionEnv(ABC):

### 2.1 LLMProvider（统一多Provider）

@dataclass
class LLMRequest:
    system: str
    user: str
    model: str | None = None
    temperature: float = 0.7

class LLMResponse:
    content: str

class LLMRegistry:
    """Provider 注册表"""
    def register(self, name: str, provider: type[LLMProvider]) -> None: ...
    def get(self, name: str) -> LLMProvider: ...
    def get_fallback(self, primary: str) -> LLMProvider: ...
```

## 5. 错误处理规范

### 5.1 错误码体系
```
格式: {DOMAIN}-{CODE}
例: HTTP-001, SKILL-003, LLM-005

Domain:
  CORE: 核心引擎错误 (CORE-001~099)
  LLM:  LLM调用错误 (LLM-001~099)
  TOOL: 工具执行错误 (TOOL-001~099)
  SKILL: Skill执行错误 (SKILL-001~099)
  CFG: 配置错误 (CFG-001~099)
  AUTH: 认证错误 (AUTH-001~099)
  SBOX: 沙箱错误 (SBOX-001~099)
  NET: 网络错误 (NET-001~099)
  DB: 数据库错误 (DB-001~099)
```

### 5.2 统一错误响应
```python
@dataclass
class ErrorResponse:
    error_code: str         # HTTP-001
    message: str            # 用户友好消息（中英双语）
    suggestion: str         # 解决建议
    trace_id: str           # 关联追踪
    details: dict | None    # 调试细节（生产环境隐藏）
```
