## 需要录制的教程

| # | 主题 | 时长 | 脚本 |
|---|------|------|------|
| 1 | 安装与首次使用 | 2min | pip install → tagent init → tagent run |
| 2 | TUI对话模式 | 1min | tagent chat → "检查 www.baidu.com" |
| 3 | 创建Skill | 3min | scaffold → validate → install → test |
| 4 | Web Dashboard | 1min | 浏览器打开 → 查看状态 |
| 5 | Visual回归测试 | 1min | tagent test visual → capture → compare |

## CLI录制命令 (准备好可直接执行)

```bash
# 教程1: 安装与首次使用
pip install test-agent
tagent init --preset minimal
tagent run "检查 www.baidu.com"
tagent report

# 教程2: TUI对话  
tagent chat
# (输入) 帮我测一下API的响应时间
# (输入) 查看上次测试结果

# 教程3: 创建Skill
tagent skill install examples/skills/demo-http
tagent skill test demo-http
tagent skill search http

# 教程4: Web Dashboard
# 浏览器打开 http://127.0.0.1:8800

# 教程5: Visual测试
tagent test visual https://example.com --name baseline
# 修改页面后:
tagent test visual https://example.com --name baseline --compare
```
