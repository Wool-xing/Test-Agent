## 4. System Prompt设计模式（纳入Agent指令设计）

### 12个可复用模板
1. 角色+目标模板
2. 约束层级模板（Hard/Soft/Forbidden）
3. 术语锁定模板
4. 输出脚手架模板
5. 验证循环模板
6. 反幻觉条款
7. 持久提醒模板
8. 工具策略模板
9. 安全段落
10. 上下文预算守卫
11. 模型Shim模板
12. Fresh-Eyes验证器Prompt

### 7个反漂移模式
1. 术语锁定 — 5-10关键术语精确注入每会话
2. 项目不变量锚定 — 不可协商模式
3. 持久记忆写入前压缩 — 关键事实在压缩前外化
4. Fresh-Eyes验证器 — 零暴露子Agent
5. 提醒指令注入 — 每次Prompt末尾
6. 单调压缩边界 — 一旦stub永不变
7. 模型特定Shim — 基础模板+每模型调整
