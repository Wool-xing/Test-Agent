## 1. 项目概况

Test-Agent 是 AI 驱动的全栈测试平台，支持自然语言描述执行测试任务。
双模式架构：AI模式（`ai/` 目录）和 CLI模式（`runtime/` 目录），汇聚于 `utils/` 共享工具层。

### 版本历程

- V1.x (v1.8.0 至 v1.21.0): 11 个 alpha 版本，16 位专家 + 32 个技能
- V2.0.0: 架构重构，259 个 runtime 文件重组为 5 层架构

## 3. 架构分析

### 当前分层（V1.x 遗留 + V2.0.0 部分重组）

```
ai/          AI 模式界面（.md 纯文档）— 16 专家 + 32 技能
runtime/     CLI 运行时引擎（混合分层）
  cli/       CLI 入口 + 命令（17 个命令）
  router/    意图路由 + 提示词
  orchestrator/ 编排引擎
  backends/  执行后端适配（7 种）
  api/       FastAPI REST
  gateway/   通知网关（9 平台）
  agent/     V2 新增：权限 + 防御
  core/      V2 新增：沙箱
  infra/     V2 新增：基础设施 14 模块
  ui/        V2 新增：TUI 10 面板
utils/       共享工具层（12 子域）
```

### V2.0.0 目标 5 层架构

- Layer 0: Runtime Core — 执行引擎、断言库、沙箱
- Layer 1: Agent Layer — LLM Provider、工具系统、Hook、Skill、权限
- Layer 2: Infrastructure — Config、Logger、Storage、Scheduler
- Layer 3: Interface — CLI REPL、TUI Dashboard、Web、API
- Layer 4: Packaging — 二进制、Docker、包管理器
