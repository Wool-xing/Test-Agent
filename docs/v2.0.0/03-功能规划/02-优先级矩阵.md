## 4. 风险矩阵

| 风险 | 等级 | 缓解 |
|------|------|------|
| LLM路由准确度不足 | 高 | 关键字表降级→LLM→默认路由 |
| 权限系统误拒合法操作 | 中 | 3级allow/deny/ask, ask为默认 |
| TUI性能不达标 | 低 | 虚拟滚动+异步更新 |
| Web安全漏洞 | 高 | OWASP Top 10全覆盖 |
| 并行执行竞态 | 中 | Worker Pool隔离+确定性重放 |

## 5. 关键依赖链

```
屎山清理(P0-01~09) → 骨架重组(P0-10) → Config/Logger(P0-11/12)

```
