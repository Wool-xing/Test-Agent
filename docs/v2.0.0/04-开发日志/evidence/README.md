# 验证证据目录

> **依据:** §补-14 验证证据要求
> **用途:** 存放每个阶段的验证证据（截图、日志、diff、测试输出等）

## 证据要求 (§补-14)

每条完成项必须附带至少一项可验证证据：

| 证据类型 | 示例 | 存放规范 |
|---------|------|---------|
| 测试输出 | pytest 运行日志 | `evidence/sprint-N-测试通过.txt` |
| 截图 | CLI 彩色输出截图 | `evidence/sprint-N-截图.png` |
| diff | 代码变更 diff | `evidence/sprint-N-diff.txt` |
| 日志 | 冒烟测试日志 | `evidence/sprint-N-冒烟.log` |
| CI 结果 | GitHub Actions 状态截图/链接 | 在验收报告中引用 |

## 命名规范

```
evidence/
  sprint-0-单元测试通过.txt
  sprint-0-覆盖率报告.txt
  sprint-1-冒烟测试.log
  sprint-1-diff.txt
  sprint-2-集成测试通过.txt
  ...
```
